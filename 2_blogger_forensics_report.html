一、調查背景與目的

JADE-ANS（翡翠網路資訊解決方案）自 2012 年起持續觀察與記錄疑似長期 SEO 污染行為對臺灣網路環境之影響。本報告針對兩個 Blogger 部落格網站進行公開網頁技術鑑識與跨網站屬性比對，藉由 HTML 結構、CSS 模板、追蹤碼與廣告指紋等技術特徵，分析其是否存在高度一致之建立流程、模板來源或操作模式。

本報告之目的在於建立可供司法或技術分析參考之關聯性線索，而非直接認定特定個人身分。

技術解讀

需特別說明的是：由於兩網站均使用 Blogger / Blogspot 平台，其 HTML 與 CSS 結構本身即可能包含大量共通元素，例如：

  • Blogger 預設模板
  • Sidebar / Footer 模組
  • 留言系統
  • RSS / Feed 元件
  • Blogger 預設 JavaScript 與 CSS

因此，高相似度結果部分可能來自平台共用結構，而非完全代表相同操作者。然而，當高比例 CSS、Class 結構與關鍵字使用模式同時出現時，仍可視為值得進一步調查之技術線索。

六、歸因分析與限制
6.1 技術觀察結果

依據本次比對結果，可觀察到：

  • 兩網站在 Blogger 模板、CSS 結構與部分內容模式上具有高度一致性
  • 相同關鍵字組與 HTML meta 欄位排列方式高度吻合
  • 存在可追蹤之 Analytics / AdSense 指紋
  • tbsexposed 與 raitanzi 之 HTML 結構 CSS 選擇器相似度達 98.3%

上述結果顯示：兩網站之間可能存在共同建立流程、相近模板來源或操作模式。

6.2 限制與注意事項

由於 Blogger 屬於統一平台，因此：

  • 相似 CSS 與 HTML 結構可能來自平台共用模板
  • 相似 Widget 與版型可能來自 Blogger 預設配置
  • 結構相似不代表法律上已完成身分認定

本報告應視為技術關聯性分析，而非單獨之法律認定證據。若需提高證據強度，建議結合：

  • Google 帳號資料及登入 IP 紀錄
  • AdSense / Analytics 綁定資訊
  • Wayback Machine 歷史版本
  • 外部圖片來源及 JavaScript / CSS 自訂來源
